Assignment Requirements Demonstration

  • Create rollover images
    ✅ Hover effects add thin dark green border and box shadow to thumbnails
  • Preload images
    ✅ All mum images are preloaded when page loads for better performance
  • Create a photo gallery using jQuery
    ✅ Click thumbnails to replace large image src and update caption text
  • Select a link with JavaScript
    ✅ Navigation links use jQuery click event handlers for selection
  • Prevent a page from following a link
    ✅ preventDefault() stops default link behavior for custom handling
  • Open a link in another window
    ✅ Click large image to open it in new window using src attribute as URL
  • Open a page within a page
    ✅ Gallery demonstrates embedding concept - can be loaded within other pages

How to Use

Gallery Navigation

Click any thumbnail image to display it as the main image. The caption will update automatically.

Hover Effects

Move your mouse over thumbnails to see the rollover effects with borders and shadows.

New Window

Click the large image to open it in a separate window for detailed viewing.

Keyboard Navigation

Use left/right arrow keys to navigate between images. Press Enter to open in new window.
